# A1-mart-website
- This is a simple E-Commerce Website for shopping products
- This website contains basic features and functionalities that are present in every E-Commerce website today

### Features
1. Responsive UI
2. Landing Page
3. Navigation Bar
4. Users - buyer & seller
5. Sign up and login
6. Authentication
7. Add products
8. Buy products
9. Comments
10. Delete Products 

### Technology Stack
1. HTML
2. CSS
3. Bootstrap 4
4. JavaScript
5. JQuery
6. Node.JS
7. Express.JS
8. MongoDB
